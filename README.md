# PaleoEurope DEM

> Reproducible pipeline for generating multi-epoch paleo-digital elevation models (paleo-DEMs) of Europe from the Last Glacial Maximum (21 ka) to the present.

## Overview

This pipeline fuses [FABDEM v1.2](https://doi.org/10.5523/bris.25wfy0f9ukibjp75gb8laq1ghq) (bare-earth land DEM) with [GEBCO 2024](https://doi.org/10.5285/1c44ce99-0a0d-5f4f-e063-7086abc0ea0f) (bathymetry) across a continuous land–sea boundary, optionally evaluates/applies an EGM2008-based geoid/undulation offset as a vertical-alignment step (validation-dependent), and then applies glacial isostatic adjustment (GIA) corrections from ICE-6G_C and ICE-7G_NA models to reconstruct topography at epochs **0 ka, 8 ka, 12 ka, and 21 ka**.

**Paper:** [https://doi.org/10.31223/X59T9W](https://doi.org/10.31223/X59T9W)
**Data (Zenodo):** [10.5281/zenodo.18804506](https://doi.org/10.5281/zenodo.18804506)  
**API Docs:** [archerby.github.io/paleoeurope-dem](https://archerby.github.io/paleoeurope-dem)

## Quickstart (5 commands, no external data needed)

```bash
# 1. Clone
git clone https://github.com/archerby/paleoeurope-dem && cd paleoeurope-dem

# 2. Create environment
conda env create -f environment.yml && conda activate paleoeurope
#    OR: pip install -e ".[dev]"

# 3. Generate synthetic test data (seed=42, deterministic)
python scripts/generate_synthetic_data.py

# 4. Run the tests on synthetic data
pytest tests/ -v

# 5. Run demo fusion on synthetic data
python scripts/run_fusion.py --config configs/default.yml --tile-list tests/data/tile_list.txt
```

Expected result: fused output tiles written to `data/output/synthetic/`.

## Installation

### With conda (recommended)

```bash
conda env create -f environment.yml
conda activate paleoeurope
pip install -e .
```

### With pip

```bash
pip install -e ".[dev]"
```

> **Note:** GDAL must be installed via conda or system package manager before pip install.  
> On Ubuntu: `apt-get install gdal-bin libgdal-dev`

### Docker (one-liner, no local install)

```bash
docker run --rm -v $(pwd)/data:/data ghcr.io/archerby/paleoeurope:v1.0 \
    python scripts/run_fusion.py --tile N51E000
```

## Hardware Requirements

| Mode | RAM | Disk | GPU |
|------|-----|------|-----|
| Synthetic demo (3×3 tiles) | 2 GB | 500 MB | Not needed |
| Single epoch, 10×10° region | 16 GB | 20 GB | Not needed |
| Full Europe, all 4 epochs | 128 GB | 2 TB | Not needed |

For full Europe runs, a Celery distributed cluster is recommended (see `distributed/`).

## Real Data Download

```bash
# Downloads FABDEM, GEBCO, ICE-6G_C, EGM2008 with SHA256 verification
python scripts/download_sample_data.py --dest data/raw/
```

All datasets are publicly available. See [ Data catalogue ](docs/data_catalogue.md) for DOIs.

## Repository Structure

```
paleoeurope-dem/
├── paleoeurope/       ← installable Python package
│   ├── fusion/        ← FABDEM+GEBCO fusion, optional EGM2008 offset
│   ├── gia/           ← ICE-6G delta-method GIA deformation
│   └── utils/         ← raster and grid utilities
├── distributed/       ← optional Celery layer (full-Europe runs)
├── notebooks/         ← Colab-compatible demo notebooks
├── tests/             ← unit + integration tests
│   └── data/          ← synthetic test data (generated by generate_synthetic_data.py)
├── scripts/           ← CLI entry points and validation tools
└── configs/           ← YAML parameter files
```
